# substrings-top

This is a function that returns a hash which lists the substrings that were found in the provided string and dictionary along with how many times they were found.

## Usage instruction
    `subtrings(<string>, <array of words>)`

Example:

    `
    dictionary = ["below","down","go","going","horn","how","howdy","it","i","low","own","part","partner","sit"]
    substrings("Howdy partner, sit down! How's it going?", dictionary)
    `
